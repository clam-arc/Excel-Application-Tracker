# Excel-Application-Tracker
An Excel job application tracker and dashboard that organizes application status, employer responses, follow-up dates, submission methods, and weekly application activity.

  Includes:
-   Submitted and planned job applications/follow up dates
-   Calculations for data and days since application start
-   Uses the =TODAY() function to automatically update the date
-   Identifies different application statuses
-   Calculates values and rates
-   Uses charts to display data

  Excel Skills Demonstrated:
-   COUNTIF(S)
-   IF and logical conditions
-   TODAY and date-related calculations
-   Relative and Absolute cell references
-   Dropdown lists
-   Conditional Formatting
-   Sorting/Filtering
-   Data analysis and cross-sheet formulas

# Workbook-Tabs
  Application Process Tab:
- Main applications
-   Name, date, status, methods, website, outcomes

  Data Tab:
- Background details
-   Total applications, rejections, active, pending, online vs in-person, response rate, weekly submission amounts
- Counts and percentages of data
- Additional Information
- Data Key
- Charts
